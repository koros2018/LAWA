## Sprint 4 详细产出

### 数据层
- `src/models/companion.py` — CompanionSession / CompanionMessage / CompanionVocabulary 三个模型

### 服务层
- `src/services/correction.py` — CorrectionEngine 实时纠错引擎（语法/词汇/表达/文化提示，中英双语）
- `src/services/vocabulary.py` — VocabularyService 生词提取 + SM-2 简化版间隔复习算法

### 场景库
- `src/data/scenarios_en.json` — 10个英文场景（咖啡厅/餐厅/机场/面试/酒店/购物/看病/商务/社交/学术）
- `src/data/scenarios_zh.json` — 10个中文场景（同上覆盖）

### Agent 层
- `src/agent/companion_agent.py` — CompanionAgent 核心Agent
  - start_session: 创建会话 + 场景角色扮演 + AI开场白
  - send_message: 对话 → 并行纠错 + LLM回复 + 生词提取
  - end_session: 总结报告 + 学习建议 + 金币结算
  - list_scenarios / list_sessions / get_session

### API 层
- `src/routes/companion.py` — 7个端点:
  - GET /scenarios — 场景列表
  - POST /session/start — 启动会话
  - POST /session/message — 发送消息
  - POST /session/end — 结束会话
  - GET /session/{id} — 会话详情
  - GET /sessions — 用户会话列表
  - POST /correct — 独立纠错

### 集成
- `main.py` 注册 companion_router
- `routes/__init__.py` 导出 companion_router
- `models/__init__.py` 导出 CompanionSession/Message/Vocabulary

## 架构亮点
- 纠错与回复**并行执行**，减少延迟
- SM-2 算法驱动间隔复习
- 场景化角色扮演（AI扮演店员/医生/面试官等）
- 金币自动累计（每轮对话+1币）
- 上下文窗口管理（保留最近15轮）
