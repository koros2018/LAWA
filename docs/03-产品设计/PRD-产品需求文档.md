## 1. 产品概述

### 1.1 产品定义
LAWA是一个非商业性的全球化语言交流智能体平台。用户在这里学习语言（中文/英文）、帮助他人学习自己的母语、通过金币经济系统获得激励——让语言学习像玩游戏一样自然且有回报。

### 1.2 核心价值主张
> **传统语言学习：** 报课（贵）→ 没人陪练（孤独）→ 学了没用（动力不足）
> **一般语言交换App：** 找不到人（冷启动）→ 尬聊（无指导）→ 沉默匹配（体验差）
> **LAWA：** AI评估(免费) → 金币激励(好玩) → AI伴读(随时) → 真人互助(真实)

### 1.3 用户故事
> **小明（中文母语者，CET-6，想练英文）：**
> "我注册了LAWA，做了英文水平测试→B2。AI给了学习计划。我每天学30分钟赚3个金币，登录赚10个，一周攒了100多金币。然后用金币找人帮我看英文邮件，一个加拿大小姐姐帮我改了，还帮我解释了为什么这么写。我也帮学中文的外国朋友改作文赚金币。像玩游戏一样，不想停。"
>
> **Emma（英文母语者，学中文2年）：**
> "我在找免费的中文语伴，LAWA可以赚金币请人帮我改作业。AI老师24小时在线，帮不上忙的时候AI立刻顶上。最重要的是——免费，不需要爸妈信用卡。"

## 3. 核心流程设计

### 3.1 用户主流程（LAWA版）
```
注册 → 选择学习语言（中文/英文） → 注册即送1000金币
  │
  ▼
CEFR/HSK水平测试 → 测试分析报告 → 水平等级锚定
  │
  ▼
画像收集 → 个性化学习规划 → 每日任务
  │                              │
  │                    ┌─────────┴──────────┐
  │                    │                    │
  │               AI伴读对话              语伴互助
  │              (赚金币中…)           (花金币/赚金币)
  │                    │                    │
  │                    └─────────┬──────────┘
  │                              │
  ▼                              ▼
  └──────── 每日金币结算 + 排行榜更新 ← ─┘
             │
             ▼
          继续学习 / 帮助他人 / 看排名
```

│  连续学习30天 ───────  +500金币（成就奖励）              │
│  邀请新用户 ────────  +500金币（邀请人+被邀请人都有）    │
│                                                         │
└─────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────── 金币流出 ───────────────────────┐
│                                                         │
│  每日自动消耗 ─────────  -10金币                         │

## 4. 功能详细说明

### 4.1 水平评估模块

#### 4.1.1 英文评估（继承ELA）
- CEFR标准 A1-C2
- 自适应语法测试 + 阅读理解 + 写作评估
- LLM动态出题 + Majority Voting评分

#### 4.1.2 中文评估（新增）
- HSK标准 1-6级
- 测试维度：汉字识别、词汇、语法、阅读、写作
- 技术方案：LLM生成中文测试题 + 中文评分rubric
- 初始等级快速锚定（10分钟内）

| HSK级别 | 词汇量 | CEFR对标 | 学习者类型 |
|---------|-------|---------|-----------|
| HSK 1 | 150 | A1 | 初学者 |
| HSK 2 | 300 | A2 | 基础 |
| HSK 3 | 600 | B1 | 中级 |
| HSK 4 | 1200 | B2 | 中高级 |
| HSK 5 | 2500 | C1 | 高级 |
| HSK 6 | 5000+ | C2 | 精通 |

### 4.2 金币系统详细设计

#### 4.2.1 账本模型
```python
class CoinTransaction:
    id: UUID
    user_id: UUID
    type: str  # register | login | study | help | consume | reward | penalty
    amount: int  # 正为收入，负为支出
    balance_before: int
    balance_after: int
    related_user_id: UUID | None  # 互助对手方
    description: str
    created_at: datetime

class CoinDailySummary:
    user_id: UUID
    date: date
    coins_earned: int
    coins_spent: int
    net_change: int
    ending_balance: int
    study_minutes: int
    helped_count: int  # 帮助他人的次数
```

#### 4.2.2 防刷机制
```
1. 单日学习奖励上限: 12金币（2小时学习）
2. 登录奖励: 每24小时一次
3. 互助交易:
   - 最低交易额: 1金币
   - 单日最高获得: 50金币（防止刷单）
   - 交易冷却: 同一用户对 10分钟冷却
4. 异常检测:
   - 日增长 > 200金币 → 标记审核
   - 批量注册识别 → IP+设备指纹
```

### 4.3 跨语言互助模块

#### 4.3.1 异步互助（MVP可做）
```
用户发帖:
  - 内容: 一段需要校对/润色的文字（中或英）
  - 悬赏金币: 用户自定
  - 标签: #英文校正 #中文校正 #翻译 #发音

其他用户接单:
  - 提交修改+解释
  - 发帖人选择最佳答案
  - 金币自动划转
```

#### 4.3.2 实时匹配（Phase 2）
```
匹配算法:
  用户属性:
    - 母语: zh | en
    - 学习语言: en | zh
    - 水平等级: B1/B2/C1
    - 当前在线状态
    - 排队时长（优先等待久的）

  匹配逻辑:
    (A.母语 = B.学习语言) AND (B.母语 = A.学习语言)
    → 配对成功
    → 各自目标水平差值 ≤ 1级（防止水平悬殊）
```

### 4.4 排行榜模块

#### 4.4.1 排行榜类型
| 类型 | 周期 | 计算方式 | 展示 |
|------|------|---------|------|
| 日榜 | 每日 | 当日结余金币 | 个人+全平台Top100 |
| 周榜 | 每周日 | 周总金币增长 | Top10 |
| 月榜 | 每月末 | 月总金币增长 | Top10 |
| 学习时长榜 | 每周 | 学习总分钟数 | Top10 |
| 帮助榜 | 每周 | 帮助他人次数 | Top10 |

#### 4.4.2 推送机制
```
每日00:05 Cron计算
  → Top100结果入Redis Sorted Set
  → 首页排行榜面板自动刷新
  → Top3全平台公告推文
  → Top11-100用户站内通知
```
